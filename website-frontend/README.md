# 软通动力官网前端项目

基于设计文档实现的软通动力企业门户网站前端项目。

## 项目简介

这是一个现代化的企业门户网站，采用纯 HTML/CSS/JavaScript 技术栈实现，无需 Node.js 环境即可运行。项目展示了软通动力的核心业务、产品服务、成功案例和企业新闻。

## 技术特点

- ✅ **纯前端实现**：无需 Node.js 环境，可直接在浏览器中运行
- ✅ **响应式设计**：支持桌面、平板、移动端多种设备
- ✅ **组件化架构**：模块化的代码结构，易于维护和扩展
- ✅ **性能优化**：懒加载、防抖节流、平滑滚动等优化措施
- ✅ **无障碍访问**：语义化 HTML，支持键盘导航
- ✅ **SEO 友好**：完善的 Meta 标签和语义化标签

## 核心功能

### 1. 多级导航菜单
- 支持三级层级的下拉菜单
- 桌面端鼠标悬停展开，移动端点击展开
- 响应式导航，移动端抽屉式菜单

### 2. 轮播横幅
- 自动轮播（每 5 秒切换）
- 手动切换（左右箭头按钮）
- 指示器点击跳转
- 鼠标悬停暂停自动播放

### 3. 产品推荐
- 网格布局展示核心产品
- 卡片悬停效果
- 响应式列数调整

### 4. 搜索功能
- 全站搜索（产品、新闻、案例）
- 实时搜索结果展示
- 关键词高亮显示
- 防抖优化

### 5. 响应式布局
- 移动设备：< 768px
- 平板设备：768px - 1024px
- 桌面设备：1024px - 1440px
- 大屏设备：> 1440px

## 快速开始

### 方式一：直接在浏览器打开

1. 下载项目文件
2. 直接打开 `index.html` 文件即可

### 方式二：使用本地服务器（推荐）

```bash
# 进入项目目录
cd website-frontend

# 使用 Python 启动本地服务器（Python 3）
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 在浏览器访问
# http://localhost:8000
```

### 方式三：使用 VS Code Live Server

1. 安装 VS Code 插件：Live Server
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 自定义配置

### 修改数据

编辑 `js/data.js` 文件来修改：
- 轮播图内容
- 产品推荐列表
- 成功案例
- 新闻资讯

### 修改样式

编辑 `css/global.css` 中的 CSS 变量：
```css
:root {
    --primary-color: #0066CC;      /* 主色调 */
    --primary-dark: #0052A3;       /* 主色调（深色） */
    --text-primary: #333333;       /* 主文本颜色 */
    --container-width: 1200px;     /* 容器最大宽度 */
    /* ... 更多变量 */
}
```

### 添加新页面

1. 在 `pages/` 目录下创建新的 HTML 文件
2. 复制 `index.html` 的基本结构
3. 修改主内容区的内容
4. 在导航菜单中添加链接

## 开发建议

### 代码规范

- 使用语义化 HTML 标签
- CSS 类名采用 kebab-case 命名
- JavaScript 变量采用 camelCase 命名
- 保持代码格式化和注释清晰

### 性能优化

- 图片使用 WebP 格式或压缩后的 JPG/PNG
- 使用 `loading="lazy"` 实现图片懒加载
- 避免大型 JavaScript 库，保持轻量级
- 使用防抖和节流优化事件处理

### 无障碍访问

- 为图片添加有意义的 alt 属性
- 使用正确的 ARIA 属性
- 确保键盘可访问性
- 保持良好的颜色对比度
